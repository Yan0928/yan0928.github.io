详解Linux下软件压缩包格式与安装：

##关于Linux下压缩与解压缩

  1. Linux下最常用的打包程序就是tar了，使用tar程序打出来的包我们常称为tar包。
    tar包文件的命令通常都是以.tar结尾的。
  2. tar可以在打包和解包的同时调用其他压缩程序
  3. 解压zip和rar
    1. linux下提供了zip和unzip程序，zip是压缩程序，unzip是解压程序
    2. 要在linux下处理.rar文件，需要安装RAR for Linux，可以从网上下载，但要记住，RAR for Linux 不是免费的
      1. 安装后就有了rar和unrar这两个程序，rar是压缩程序，unrar 是解压程序。

tar命令 格式： tar [选项] [参数] 文档名

例如： tar -xvf packag

-选项：

-c: 建立压缩档案
-x：解压
-t：查看内容
-r：向压缩归档文件末尾追加文件
-u：更新原压缩包中的文件

-参数：

-z：有gzip属性的
-j：有bz2属性的
-Z：有compress属性的
-v：显示所有过程
-O：将文件解开到标准输出

#####最后参数-f是必须的-f: 使用档案名字，切记，这个参数是最后一个参数，后面只能接档案名。

gzip压缩方式

1. 对于.gz结尾的文件  
    1. gzip -d all.gz
    2. gunzip all.gz
2. 对于.tgz或.tar.gz结尾的文件  
    3. tar -xzf all.tar.gz
    4. tar -xzf all.tgz

bzip2压缩方式

1. 对于.bz2结尾的文件   
    1. bzip2 -d all.bz2 
    2. bunzip2 all.bz2 
2. 对于tar.bz2结尾的文件   
    3. tar -xjf all.tar.bz2 
    4.

compress压缩方式

1. 对于.Z结尾的文件   
    1. compress all.Z 
    2. uncompress all.Z 
2. 对于.tar.Z结尾的文件   
    3. tar -xZf all.tar.z 
    4.

##通过命令行方式安装

(1) 安装与卸载命令详解

  1. sudo apt-get install packagename //命令安装软件
  2. apt-get -f install //安装时自动下载相关依赖包

>

* 如果我们知道我们要安装的软件的确切的名称，可以此条命令来获取和安装软件。
* apt-get是一条linux命令，适用于deb包管理式的操作系统
如ubuntu，主要用于自动从互联网的软件仓库中搜索、安装、升级、卸载软件。
* apt-get命令一般需要root执行，所以一般跟着sudo命令。

(2) dpkg命令详解：Debian Package的简写

  1. dpkg -i packagename.deb // 安装下载好的.deb软件

>

* 后缀deb是Ubuntu的可安装格式，可双击安装也可用dpkg命令
* 格式：dpkg [<选项>]  [参数]  软件全名
* dpkg命令中 选项-i 是install的含义，参数 packagename.deb 是软件名

(3) 后缀为bz2的压缩包，表明是以使用bzip2压缩格式的压缩包

>

1. 先要进行解压操作。tar -zxvf FileName.tar.bz2 解压后的如果是一个文件目录，那么cd目录，看README或者INSTALL文件，里面有详细说明，包括安装、编译、配置等等
2. 解压后分为源代码安装包和可执行安装包两种
    1. 源代码安装包：
        1. 通过执行./configure来进行配置
        2. 执行make来进行编译
        3. 执行make install来进行安装
    2. 可执行安装包
        1. cd进目标目录下 ./可执行文件名
